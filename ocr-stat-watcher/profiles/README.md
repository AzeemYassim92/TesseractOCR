# Character Profiles

Profiles let each character use different HP/MP thresholds without changing OCR code.

## Run

```powershell
python main.py
python main.py --main
python main.py --gins
python main.py --profile gins
```

## Hotkeys

- `Q`: toggle loot spacebar spam.
- `W`: toggle AutoTargeting.
- `E`: run warehouse ctrl+double-click sequence.
- `R`: pause/resume OCR `z` and `x` actions.
- `T`: print the active profile thresholds.
- `L`: stop the active warehouse run.

## Profile Files

Edit JSON files in this folder to tune character-specific thresholds.
